1) Скачать с Github проект
2) Поднять базу PostDB из бекапа.
Открыть SSMS, лкм по пункту Базы данных -> Восстановить базу данных
Источник - Устройство, по трём точкам открыть форму выбора, тип носителя - Файл. Через Добавить добавляем файл бекапа -> ОК.
База данных должна быть PostDB, нажимаем ОК.
3) Как работает выбор сервера, на котором находится база.
Требуется в основном окне выбрать в шапке ссылку(она только одна), после загрузки новой страницы выбирается строка с нужным сервером.
После этогонажать на ссылку "Домашняя страница", чтобы вернуться назад и загрузить данные из таблицы.
