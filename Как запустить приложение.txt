1) Скачать с Github проект
2) Поднять базу PostDB из бекапа.
Открыть SSMS, лкм по пункту Базы данных -> Восстановить базу данных
Источник - Устройство, по трём точкам открыть форму выбора, тип носителя - Файл. Через Добавить добавляем файл бекапа -> ОК.
База данных должна быть PostDB, нажимаем ОК.
